ARIA 1.1

div with role combobox

text input with role textbox

popup listbox
controlled by text input

<div role="combobox">
 <input type="text"
 aria-controls="lb">
  <ul role="listbox"
 id="lb">...
  </ul>
</div>
  • Historically, screen readers have expected combobox to be focusable input.
  • 1.1 pattern changed combobox into a composite container.
  • Except in interaction mode, no screen reader supports this pattern.
  • Also has unresolved naming issues.
